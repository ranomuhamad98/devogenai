Scale on hover
How to Use:
Setting this option will scale tilt element on hover. 
$('.js-tilt').tilt({
   scale: 1.2
})
Disable Y axis
The most basic usage:
Setting this option will disable the Y-Axis on the tilt element. 
$('.js-tilt').tilt({
   axis: x
})
Disable X axis
The most basic usage:
Setting this option will disable the X-Axis on the tilt element. 
$('.js-tilt').tilt({
    axis: Y
})
